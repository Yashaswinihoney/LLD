02 · Design Decisions
Storage
ConcurrentHashMap
Lock-striped → no global lock. Thread-safe at segment level.
Expiry Clock
currentTimeMillis()
Wall-clock is correct for TTL. Monotonic not needed here.
Lazy Remove
On get() access
Zero write-path overhead. Instant eviction on hot keys.
Active Sweep
ScheduledExecutor
Reclaims memory for cold keys that are never read again.
Atomic Remove
remove(key, entry)
CAS prevents evicting a freshly re-put entry in a race.
TTL Granularity
Milliseconds
Fine-grained for OTPs, sessions, rate-limiting windows.
